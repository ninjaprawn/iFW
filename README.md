Building
----------
First, either download the repo or clone it:
```bash
git clone https://github.com/ninjaprawn/iFW.git
```

Then install with CocoaPods:
```bash
pod install
```

Libraries/Frameworks
----------

iFW uses the following libraries/frameworks:
- Alamofire
- SwiftyJSON
- SwiftRSS
- Realm
